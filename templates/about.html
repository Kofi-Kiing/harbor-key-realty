ABOUT THIS PROJECT

A fictional brokerage with a real, working stack.

Harbor & Key Realty is a development sandbox. The names, addresses, prices, contact details, and listings are invented for testing.

WHAT YOU CAN TEST

Use it as a clean starting line.

The project gives you realistic real estate pages without requiring a CMS or production database. Listing data lives in one JSON file, so it is easy to replace, generate, scrape, or connect to another system.

You can test SEO metadata, analytics events, UI experiments, web automation, contact integrations, deployment hooks, API consumers, or a future admin dashboard.
